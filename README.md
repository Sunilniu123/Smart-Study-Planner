# Smart Study Planner

Frontend-only productivity web app.

## Features

- Task Management
- Pomodoro Timer
- Analytics Dashboard
- Calendar
- Dark Mode
- Voice Commands
- Notifications
- Gamification
- Offline Support
- PWA

## Run

1. Install VS Code
2. Install Live Server Extension
3. Open Project Folder
4. Right Click index.html
5. Open with Live Server
