# 📚 Smart Study Planner

A modern and responsive frontend-only productivity web application built using HTML, CSS, and JavaScript. This project helps students organize tasks, improve focus, track productivity, and manage study schedules efficiently.

---

# 🚀 Features

## 🎨 UI / UX
- Modern Dashboard
- Glassmorphism UI
- Dark / Light Mode
- Multiple Theme Colors
- Responsive Design
- Animated Background
- GSAP Animations
- Loading Screen

---
